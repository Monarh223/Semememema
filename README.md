# MAX QR Bot (Playwright)

Telegram-бот: по команде `/qr` открывает web.max.ru в headless-браузере, присылает скриншот с QR. После сканирования QR в приложении MAX бот сам читает `localStorage` и присылает готовый блок для переноса сессии в другой браузер.

## Установка

```bash
pip install -r requirements.txt
playwright install firefox
```

Создай `.env` с `TELEGRAM_BOT_TOKEN` (от @BotFather). Для учёта пользователей добавь `BOT_ADMIN_IDS=<твой_telegram_id>`.

## Запуск

```bash
python bot.py
```

## Учёт пользователей и админка

Бот хранит БД (SQLite, `bot.db`) и выдаёт доступ только разрешённым. Первый вход — запрещён, пока админ не разрешит.

**Настройка админов** — в `.env`:
```env
BOT_ADMIN_IDS=123456789
```
Узнать свой ID: @userinfobot. Несколько админов через пробел: `BOT_ADMIN_IDS=111 222 333`.

**Команды админа** (`/admin`):
- `/admin` — панель с кнопками
- `/admin stats` — сводка: пользователи, допущенные, всего токенов
- `/admin users` — список пользователей с счётчиком токенов
- `/admin allow <user_id>` — разрешить доступ
- `/admin deny <user_id>` — запретить

## Использование

1. Отправь боту `/qr`.
2. Получишь картинку с QR — отсканируй в MAX: Профиль → Устройства → Войти по QR-коду.
3. После сканирования бот пришлёт готовый блок. Скопируй и выполни его в консоли (F12) в том браузере, куда хочешь перенести сессию.
