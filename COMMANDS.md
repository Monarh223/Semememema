# MAX TOKEN — полный список команд и кнопок

## Команды (текстовые)

### Общие (личка и группа, где есть доступ)

| Команда | Где | Кто | Описание |
|--------|-----|-----|----------|
| **/start** | Личка, группа | Все (в группе — владелец/саппорты) | Открыть меню. В группе: зарегистрировать группу как саппорт (если первый раз) или показать приветствие и команды группы. |
| **/qr** | Личка, группа | С доступом / саппорт | Запустить получение QR-кода для сканирования в MAX. После скана бот пришлёт .txt с блоком сессии. |
| **/link_phone** | Личка, группа | С доступом / саппорт | Привязка по номеру: ввод номера → SMS-код → (при 2FA пароль) → бот пришлёт .txt с блоком. |
| **/tokens** | Личка, группа | С доступом; в группе — владелец или саппорт с «доступом к токенам» | Показать количество сохранённых токенов и кнопки «Выдать 1 / 3 / 5». |
| **/tokens N** | Личка, группа | То же | Сразу выдать N токенов (например `/tokens 3`) в виде .txt файлов. |
| **/check_proxy** | Личка, группа | С доступом | Проверить IP через браузер (тот же прокси, что и для /qr). |

### Только в группе (владелец группы)

| Команда | Описание |
|--------|----------|
| **/addsupport** | Добавить саппорта. Варианты: отправить в **ответ на сообщение** пользователя или **/addsupport USER_ID** (числовой ID). |
| **/addsupport USER_ID** | Добавить саппорта по числовому ID Telegram. |
| **/delsupport USER_ID** | Убрать саппорта из группы (лишить возможности пользоваться ботом в этой группе). |
| **/supportlist** | Список саппортов, статистика «кто сколько токенов сделал», кнопки: забрать доступ из группы, выдать/забрать «доступ к токенам». |
| **/groupstats** | Статистика по группе: кто сколько токенов сделал (видят владелец и саппорты). |

### Админка (только админы, обычно в личке)

| Команда | Описание |
|--------|----------|
| **/admin** | Открыть админ-панель (кнопки: Статистика, Пользователи, Прокси) и справку по подкомандам. |
| **/admin stats** | Сводка: всего пользователей, с доступом, успешных токенов. |
| **/admin users** | Список пользователей (до 50) с флагом доступа и счётчиком токенов. |
| **/admin allow USER_ID** | Разрешить пользователю доступ к боту. |
| **/admin deny USER_ID** | Запретить доступ (заблокировать). |
| **/admin proxies** | Список прокси из proxies.txt (номера, строки, статус). |
| **/admin proxy_add ip:port:user:pass** | Добавить прокси (можно пачкой, по строке или через запятую). |
| **/admin proxy_del N** | Удалить прокси по номеру N из списка. |
| **/admin proxy_del строка** | Удалить прокси по точной строке (например ip:port:user:pass). |
| **/admin proxy_refresh** | Обновить прокси из Proxy-Seller (если настроен proxy_seller_user_api). |

## Кнопки главного меню (инлайн)

Появляются после /start или по нажатию «📋 Меню» / «❓ Помощь» в чате.

| Кнопка | Действие |
|--------|----------|
| **📞 По номеру** | Запуск привязки по номеру (то же, что /link_phone). |
| **📱 QR-код** | Запуск QR-флоу (то же, что /qr). |
| **🎁 Мои токены** | Экран токенов: сколько есть, кнопки «Выдать 1 / 3 / 5». |
| **❓ Помощь** | Текст-гайд по использованию бота. |
| **↻ Обновить меню** | Обновить сообщение с меню (то же меню заново). |
| **💬 Поддержка** | Ссылка в поддержку (SUPPORT_LINK). |
| **⚙️ Админ-панель** | Только у админов: переход в админ-панель. |

## Кнопки в процессе привязки по номеру

| Этап | Кнопка | Действие |
|------|--------|----------|
| Ввод номера | **❌ Отмена** | Выйти из флоу, не вводить номер. |
| Ввод кода из SMS | **Кода нет — пропустить** | Отменить ожидание кода (прервать авторизацию). |
| Ввод кода из SMS | **❌ Отмена** | То же: отменить ожидание кода. |
| Ввод пароля 2FA | **❌ Отмена** | Отменить ввод пароля. |
| После ошибки | **📞 Повторить / другой номер** | Вернуться к выбору «По номеру» (можно снова ввести номер /link_phone). |

В чате при ожидании кода/пароля также можно написать текстом: **отмена**, **cancel**, **не могу отменить** и т.п. — бот отменит ожидание и напишет «Авторизация отменена…».

| Кнопка | Действие |

## Кнопки админ-панели

| Кнопка | Действие |
|--------|----------|
| **📊 Статистика** | Сводка (пользователи, доступ, токены). |
| **👥 Пользователи** / **👥 Список пользователей** | Список пользователей с пагинацией. |
| **🧩 Прокси** | Список прокси из proxies.txt. |
| **◀ Назад** | Назад в меню админки. |

| Кнопка | Действие |
|--------|----------|

| **✅ Разрешить USER_ID** | Выдать доступ. |

| Кнопка | Действие |
|--------|----------|
| **🔄 Обновить прокси** | Синхронизация с Proxy-Seller (/admin proxy_refresh). |

## Кто что может

- **Личка без доступа** — только сообщение «нет доступа»; /start показывает это.
- **Личка с доступом** — /start, /qr, /link_phone, /tokens, /check_proxy, всё меню кроме админки.
- **Админ** — всё то же + /admin и кнопка «Админ-панель».
- **Группа: владелец** — всё то же + /addsupport, /delsupport, /supportlist, /groupstats, управление саппортами и доступом к токенам, обнуление статистики.
- **Группа: саппорт** — /start, /qr, /link_phone, /tokens (если владелец выдал «доступ к токенам»), /groupstats, меню без админки (админка только если пользователь админ глобально).
